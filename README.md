## `✦ Experience`

<br>

**Full Stack Engineer — Yeb Inteligência de Mercado**

Delivered backend-heavy, full stack features across multiple enterprise B2B products in the agribusiness sector. Worked directly with product and business teams on technical refinement, requirements definition, and solution design.

- Designed and maintained **RESTful and GraphQL APIs** consumed by enterprise clients in the energy sector
- Built event-driven integrations using **Kafka and RabbitMQ**, enabling decoupled, high-throughput data pipelines
- Implemented **authentication and authorization** layers with Spring Security and JWT across multi-tenant platforms
- Led backend architecture decisions following **DDD, Clean Architecture, and SOLID** principles
- Set up and maintained **CI/CD pipelines** (Jenkins, Docker) for cloud environments (AWS, Azure)
- Contributed to frontend surfaces with **React, Vue.js, and TypeScript**, owning end-to-end feature delivery
- Applied **TDD and integration testing** with JUnit, Jest, Testcontainers, and Playwright to ensure production reliability

Enterprise clients: **Raízen &nbsp;·&nbsp; Cerradinho Bioenergia**
